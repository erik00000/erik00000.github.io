學習如何在windows中清楚DNS緩存

發表於2024-01-03|更新於2024-01-03|DNS
|閱讀量:

在有些時候，如果更新了自己域名的 DNS 記錄，但是由於緩存的原因，解析的還是舊的 DNS。因此，我們需要在自己的電腦清理下自己的 DNS 緩存。這篇教程我來教大家在 Windows 中如何清理 DNS 緩存

準備材料
Windows 電腦
部署步驟
在搜索框中，輸入 cmd，然後點擊“以管理員權限運行”

輸入以下命令
ipconfig /flushdns
如出現以下提示，即可代表已清理 DNS 緩存
