trollstore巨魔2安裝教程：小白也能輕松搞定

發表於2024-01-09|更新於2024-01-09
|閱讀量:

根据搜索结果，TrollStore2（巨魔商店2）的最新版本已经更新，支持iOS 16.5.1 - 16.6.1（16.7）[1]。具体的发布日期和版本号没有在搜索结果中明确给出。如果你想获取最新的版本信息，建议直接访问TrollStore2的官方GitHub页面[1]，或者订阅相关的YouTube频道[5]，以便获取最新的视频更新。

我们都知道安卓可以随意下载 .apk 文件进行安装，但是 iPhone 因为安全问题是不允许这样的，iPhone 只能从 App Store 下载和安装应用。

但是如果 iPhone 手机安装巨魔成功后，它可以让我们安装任意的 IPA 应用。

原理：非越狱，它可以让我们给 .ipa 应用永久签名，然后就可以安装和使用。

巨魔商店2 已经发布有一段时间了，截至发文章时，支持 IOS 版本如下：

15.5 - 16.6.1（已经支持）
17.0（未来支持，目前没有安装方法）

安装教程
IOS15.5-15.6.1
支持 iPhone XR - 13 Pro Max（A12 - A15）所有型号，直接点击

不支持其他 IOS 版本；
不支持 A8X - A11 处理器；
不支持 iPhone6s - X；

IOS15.5-16.6.1
支持 A12+ 处理器

不支持 A8X - A11 处理器；
不支持 iPhone6s - X；

下面操作均需要开启魔法上网！

IOS16 系统必须开启开发者模式（设置 -> 隐私与安全性 -> 开发者模式）

电脑端 戳我 下载 misaka 的 .ipa 软件包，地址：

使用数据线连接手机至电脑，使用签名工具（爱思助手、轻松签、牛蛙助手等工具都可，不会自行百度）自签 misaka 的 .ipa 安装包，并安装到 iPhone 手机中；

iPhone 打开 misaka 软件，点击 Continue，点击 close，点击 Agree，软件底部切换到 Packages，点击左下角四方形图标。（如果提示 KFD Mode，选择 Yes）；

点击 install Trollstore，等待一下，反应可能会有点慢，也有可能重启，重启需要重复这一步；

点击 Tips ，如果弹窗提示 done，不要点 OK！不要点 OK！不要点 OK！ 直接关机。
同时按住任一音量按钮和侧边按钮，直到出现关机滑块，滑动关机即可）；

开机，开启魔法上网，打开手机上 提示APP（没有的话去 App Store 下载），点击 install TrollStore，开始安装巨魔2 ；

安装完成，返回桌面，打开桌面巨魔图标，自动安装 ldid，如果没有，点击底部Setting，点击 install ldid；

设置巨魔，往下滑动 点击 install Persitence Helper，选择 Tips，安装巨魔永久性助手，万一巨魔白图标或者打不开，

还可以打开提示APP，重新安装巨魔（这步必须做）；

设置巨魔简易安装软件方式：往下滑动，开启 URL Scheme Enabled，选择 Rebuild Now，刷新缓存。
这一步开启是方便安装 IPA 软件。

如果要安装软件，在资源网盘直接点击 TrollStore 直接就会跳转安装，而不是打开其他应用！！！
