谷歌云服务器 Google Cloud 谷歌云 简介

Google Cloud 是 Google 提供的免費雲端伺服器，在 2024 年仍張開雙臂歡迎用戶。 Google Cloud 在 2024 年提供 300 美元的試用積分。 如果您提供有效的電子郵件地址，您將額外獲得 100 美元，使試用成為可能。 信用總額為 400 美元。 優惠有效期為 90 天，之後無需擔心，您可以無限制地申請延期！ GCP是目前最貼心的雲端平台，沒有任何限制，可以讓您嘗試各種類型的雲端主機。

Google Cloud 谷歌云(Google云平台)？

Google雲端平台（Google Cloud Platform），通常簡稱為GCP，是Google提供的一種雲計算服務，將Google的基礎架構、資料分析和機器學習工具提供給其他公司和開發者。
Google雲端平台包含一系列托管在Google基礎設施上的雲計算服務，包括各種主流的運算、儲存、數據分析和機器學習服務，還有專為雲端工作提供的管理工具。公司或開發者可以透過Google雲端平台達成高效、靈活且具規模優勢的雲端解決方案。
它是業界公認的領先雲服務商之一，與Amazon 的AWS和微軟的Azure並駕齊驅，提供在全球的數據中心中執行應用程式的能力。

Google雲端平台（GCP）提供哪些雲計算服務？

Google雲端平台（GCP）提供了一系列的雲計算服務，包括但不限於以下部分：
計算服務：例如 Compute Engine（虛擬機械，硬碟和網絡），Kubernetes Engine（容器化應用管理）
儲存服務：例如 Cloud Storage（雲存儲服務）
大數據服務：例如 BigQuery（全面的大數據分析服務）
人工智能服務：例如 Cloud AI Platform（AI開發、建模與訓練平台）
網絡服務：提供多種虛擬網路相關功能，如負載均衡和云端VPN等。
管理工具：為雲端工作提供的管理工具，如 Google Cloud Console、雲端Shell 和 Google Cloud SDK。
此外，GCP也提供各種資料庫服務，資料傳輸服務，以及開發、身份與安全相關工具等。它實際上是一個全方位提供雲端解決方案的平台。

谷歌雲端免費試用90天300美金試用金

新的Google Cloud Platform（GCP）用户可以享受300美金的免费试用金，并且此优惠的试用期为90天。
用户在注册试用期后可获得300美金的信用，可用来支付对GCP各种服务的使用费用，包括虚拟机、数据库、数据分析以及其他各种服务。这为新用户提供了一个很好的机会对GCP服务进行全面的体验和评估，无需承担经济风险。
值得注意的是，这300美金的试用金需要在90天内使用完，如果过了这个期限，剩余的试用金将会被回收，不会转到正式账户。在试用期结束后，你需要设定支付方式以便继续使用GCP的服务。
在这期间，你也可以随时查看你的账户余额和使用情况，以控制你的花费。同时，GCP也会发送邮件提醒你试用金使用情况和余额。

正式開始申請

造訪Google CloudGoogle雲端的網站，cloud.google.com


在選擇國家部分，选择你ip相关的地区， 這樣做確保了你的代理IP和你登記的國家/地區是相符的，可以避免一些可能的問題。
在選擇「哪一項與您的組織或需求最相符」這部分，GCP通常會提供幾個選項供您選擇。 這些選項可能包括，但不僅限於：

個人需求：如果你是以個人使用者身分註冊，並計畫使用GCP進行學習、研究或個人的專案開發，你應該選擇「個人需求」這個選項。
商業需求：如果你代表一家公司或企業進行註冊，你應該選擇「商業需求」這個選項。 這可能包括你打算在GCP上託管你的業務工作負載，或者你打算使用GCP進行商業級的應用開發等。
教育需求：如果你是代表一個教育機構進行註冊，或者你打算用GCP進行教育或研究目的，你應該選擇「教育需求」這個選項。
這些選項會根據你的具體情況進行選擇，確保最大程度複合你的使用目的和需求。

添加付款资料

過程中名字全是用的英文

對的，為了驗證你的身份以及確保你有足夠的資金來支付可能產生的費用，Google Cloud Platform（GCP）在註冊過程中通常需要你提供一個有效的信用卡資訊。
在填寫信用卡資訊時，你需要提供信用卡的卡號、到期日、安全碼，以及你的個人信息，如姓名和地址等。 在這過程中，GCP會透過一個小額扣費（通常是1美元）來驗證你的信用卡是否有效。 這筆費用在成功驗證後會退回。
選擇個人帳戶類型是非常好的，尤其如果你是作為個人用戶註冊使用GCP。
在這個過程中，你無需擔心你的個人資料的安全問題。 Google有非常強大的資料安全策略和合規框架來確保你的個人資訊和付款資訊的安全。

在验证信用卡信息时，Google Cloud Platform（GCP）会尝试从你的信用卡上扣取一个小额数目——如你所述，如果你是在美国，这个数字通常是1美元；在香港，则是HKD8.00。
这个验证扣费的目的是为了确认你所提供的信用卡信息是正确并且有效的。这个扣费会在验证完成后返还给你，所以你并不需要为这个扣款支付实际的费用。
扣款成功后，GCP会将一条确认信息发送到你的手机银行APP上或者是你的邮箱，以确认你的付款信息。这条信息会包含一个交易参考码，通常是以“GOOGLE*”开始，后面跟随一串特定的字符和数字。这个参考码可以被用作验证你的支付信息及交易。

Google Cloud Google雲端 帳號關聯設置

登入你的新Google Cloud帳號，然後在左側選單中選擇「結算」選項。 在對應的頁面中，找到並點擊“帳號管理”，然後在右上角點擊“顯示資訊面板”。
接下來，你需要新增一個主帳號。 在 “我的結算帳號” 資源的 “新增主帳號和角色” 部分，鍵入你舊Google Cloud帳號的電子郵件地址，這將作為新的主帳號。 在角色選擇部分，選擇 “Billing > Billing Account Administrator” ，這將賦予該帳號管理員權限，從而可以全面管理和控制結算資訊。
儲存你的變更後，你可以看到新帳號已成功新增為主帳號。
最後一步是將新Google Cloud的結算帳號重新命名為“google90”，以便於你在後續中更容易識別和管理。 這樣，新谷歌雲端的初步設定就完成了。

主账号操作流程

登录老的Google Cloud账户。
在左侧菜单栏，点击“结算”选项。
再点击“账户管理”，此时平台上会显示出你所有的结算账户。
点击“转至关联的结算账户”选项。你会看到所有关联的账户列表。
在这个列表中，你应该能看到你刚才在新账户上创建并命名为“google90”的结算账户。注意，在这个步骤，我们先不选择这个账户。
点击左侧菜单栏最下方的“账户管理”选项。
在“老谷歌云”账户上，找到名为“My First Project”的项目，点击“操作”，然后点击“更改结算信息”。
在“结算账户”选项中，选择你之前标记为“google90”的账户，然后点击“设置账户”来完成关联

Google Cloud確實會在第一次註冊後提供90天的免費期，這段期間使用者可以免費使用特定金額的資源和服務。 但是，如你所說，如果在這段期間內你的網站流量大或使用的資源多，這些免費的資源可能不足以支援90天。
但請放心，即使在這段期間免費資源用完，你仍然可以繼續使用Google Cloud的服務，但需要按照正常的計費標準支付費用。 另一種方法，像你提到的，就是在90天免費期結束後，可以註冊新的Google帳號併申請Google Cloud，然後繼續享受新的90天免費期。 然後你可以透過類似的方式將你的專案和資源轉移到新帳戶，以達到續約的目的。
確實，如果Google可以將免費時長從90天調整到365天，會讓很多使用者省去很多麻煩。 我們也很理解你的感受，希望Google Cloud能夠聽取使用者的回饋，不斷優化和改進他們的服務。
最後，請記住，使用雲端服務時最重要的是理解並控制你的消費情況，以便在獲得所需的服務的同時，盡可能節省成本。
